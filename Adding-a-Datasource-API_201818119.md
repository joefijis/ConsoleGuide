---
title: "RxCheck : Adding a Datasource API"
parent: "RxConsole User Guide"
nav_order: 8
---
1. Follow steps 1-3 under the section titled V*iew and add a datasource for provider validation* in this guide.

2. Select the “API” option under the *Datasource Type* dropdown.

3. Enter the API URL into the field labeled *API URL*.

4. Select the appropriate authorization type using the *Auth Type* dropdown.

5. For BASIC authorization, enter the username and password.

6. For OAUTH2 authorization, enter the Client ID, Client Secret, and Auth URL into their respective fields.

7. Press the *Save* button on the right corner of the screen to add your datasource API to the RxConsole for your state.

8. Alternatively, if you do not wish to add an API you can press the *Cancel* button to discard changes and return to the previous screen.


<div class="step-image-layout" markdown="1">
<div class="steps-col" markdown="1">

9. Verify the datasource API was successfully added by looking for the Alert that states, “Datasource Created”.
   You will also return to the Datasource screen and should see your datasource in the *List of Datasources*.
