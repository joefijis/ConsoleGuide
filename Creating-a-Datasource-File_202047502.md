1. Open Microsoft Excel on your computer.

2. Create a new blank workbook.

3. Add your desired headers in any order you prefer. The names of the possible headers are:

* First Name
* Last Name
* NPI#
* DEA#
* SL#
* OTHER#

The header names should be entered in cells A1, B1, C1, D1, E1 and F1.

Not all header names are required and any combination of headers can be used (For example, if a state administrator does not want to use the DEA number, the may elect to remove that header completely.

1. Populate the relevant information under the corresponding column.

Duplications in the First Name and Last Name column are expected and will not affect provider validation.

1. When finished building your datasource file, save the file as either a .csv (Comma Separated Values (CSV)) or .xlsx (Excel workbook).
