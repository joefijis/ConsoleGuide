1. Click on the *PDMP Maintenance Schedule* button, located on the left-hand side of the screen.

2. Click on the *Add Maintenance* button located on the top right-hand corner of the screen.

You can only create a new maintenance event if no other upcoming event is scheduled. If there’s a conflict, an alert message will appear.

1. A pop-up screen titled *Maintenance* will appear.

2. Select the start date and time by making appropriate selections in the calendar by clicking the blue calendar in the field labeled *From Date&Time*.

3. Select the end date and time by making appropriate selections in the calendar by clicking the blue calendar in the field labeled *To Date&Time*.

4. Select the reason for the maintenance event by clicking on one of the dropdown options for the field labeled, *Reason Type*.

5. Add a note into the text box labeled, *Reason Message* to add context to the maintenance event.

6. Press the *Save* button to add the new maintenance event to the maintenance schedule in the RxConsole.

**To modify an already existing maintenance event**

---

1. Click on an event listed in the *Maintenance Schedule*.

2. Modify any information in the *Maintenance* popup similar to if you were adding a new event.

3. Press the *Save* button to record the changes to the maintenance event in the RxConsole.

4. Alternatively, you can press the *Close* button to discard changes and return to the previous screen.

5. Press the *Complete* button to mark the event as completed.

<div class="steps-col" markdown="1">

6. Press the *Cancel* button to cancel the event in the maintenance schedule.
