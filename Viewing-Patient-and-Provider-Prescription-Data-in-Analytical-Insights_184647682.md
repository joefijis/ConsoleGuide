## To search by County

1. Click on the *County* radio button.

The *County* button is selected by default.

1. Click on the downward-facing arrow for the filter titled “County” to reveal a list of counties.

2. Select your County(ies) by:

   1. Scrolling through the dropdown options and checking the box, or
   2. Typing in the search bar to filter results.

3. Click the *Search* button to display prescription counts for patients based on the counties selected.

## How to search by Zip Code

1. Click on the *Zip Code* radio button.

2. Enter the zip code and the desired distance in miles around the zip code to be searched.

3. Click the *Search* button to display prescription counts for patients based on the zip code and range entered.

4. Results are displayed on the map with the user’s state highlighted green and the connected states in orange.

Click on the *Reset Filters* button to return all selections to the default values.
